Quick Setup for P5

  1. Login to GitHub.com / Make an Account
  2. Once you have signed up for, verified your account, and are logged in. Go to this address https://github.com/Townclock/p5-GitSiteTest
  3. Click on the green "Use this template" button
  4. choose a name for your project and create the repository
  5. in your project, click on sketch
  6. copy and paste in your current sketch.js file
  7. commit your change

now you have a git repo that you can add people too

  1. go to settings
  2. click on the manage access option
  3. click on the manage access option

host on github pages

  1. go to options
  2. scroll down the the github pages options
  3. select the main branch and save
  4. now you have a link to the hosted page
  5. it may take a few minutes for the page to run
